其中， 
顶点数据(Vertex data)：  
　　v  几何体顶点 (Geometric vertices)  
　　vt 贴图坐标点 (Texture vertices)  
　　vn 顶点法线 (Vertex normals)  
　　vp 参数空格顶点 (Parameter space vertices) 
元素(Elements):   
　　p 点 (Point)   
　　l 线 (Line)   
　　f 面 (Face)   
　　curv 曲线 (Curve)   
　　curv2 2D曲线 (2D curve)  
　　surf 表面 (Surface)
